# Popeye Boileplater

<img src='./popeye.jpg' />

### Installing

The first step you must follow to run the boilerplate is to install all its dependencies.

The following commands should do this for you.

```
yarn install
```

### Usage

```
  yarn start
  localhost:3000/
```
